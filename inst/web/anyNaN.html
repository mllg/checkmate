Check if an object contains NaN values

Usage

anyNaN(x)

Arguments

x
[ANY] Object to check.

Value

[logical(1)] Returns TRUE if any element if NaN.

Description

Supported are atomic types (see is.atomic), lists and data frames.

Examples

anyNaN(1:10)



[1] FALSE



anyNaN(c(1:10, NaN))



[1] TRUE



iris[3, 3] = NaN
anyNaN(iris)



[1] TRUE
