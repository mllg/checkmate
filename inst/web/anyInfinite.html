Check if an object contains infinite values

Usage

anyInfinite(x)

Arguments

x
[ANY] Object to check.

Value

[logical(1)] Returns TRUE if any element if -Inf or Inf.

Description

Supported are atomic types (see is.atomic), lists and data frames.

Examples

anyInfinite(1:10)



[1] FALSE



anyInfinite(c(1:10, Inf))



[1] TRUE



iris[3, 3] = Inf
anyInfinite(iris)



[1] TRUE
